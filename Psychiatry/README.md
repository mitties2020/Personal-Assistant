# Psychiatry

This folder will contain psychiatry notes, assessment tools, diagnostic guides, and psychopharmacology references.

Large textbooks and PDF resources are stored in Releases due to size.

## Topics to include
- Mental Status Examination (MSE)
- Risk assessment: suicide, aggression, absconding, capacity
- Emergency psychiatry protocols
- Depression, bipolar, anxiety, psychosis, personality disorders
- DSM-5/ICD-11 reference notes
- Psychotropic medication doses and monitoring
- ECT indications and consent
- CAT/EMHS forms & templates (where allowed)
